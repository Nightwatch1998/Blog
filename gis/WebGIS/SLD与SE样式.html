      # SLD和SE的区别

      # 概念

      SLD（Styled Layer Descriptor）是OGC提出的一种基于XML的样式语言，用于我WMS服务提供样式。

      SE（Symbology Encoding）是SLD的子集，明确定义了数据样式要素，SE和SLD可以配合使用。SE也可以不依赖WMS上下文，单独使用。

      # SLD/SE用途

      为地理信息数据提供样式

      • 数据类型：
        • 矢量数据，如点、线、面
        • 栅格数据，如高程数据
      • 为要素提供样式：
        • SVG/CSS 颜色
        • 线型
        • 虚线和实线
        • 图标，例如圆、方形、十字
      • 基于条件的样式
      • 过滤器
