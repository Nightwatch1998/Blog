# SpringBoot热更新

# springboot-devtool

  1. 添加maven依赖
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-devtools</artifactId>
    <optional>true</optional>
</dependency>
  1. 添加yml配置 只是监测classpath(resources)下的文件，构建后才会自动更新
# 填写配置
spring:
    devtools:
         restart:
         enabled: true
         additional-paths: src/main/java

  1. 更新代码后手动点击构建，前端请求就会刷新了 DEBUG running模式都可以

  2. 自动构建

    1. IDEA中勾选：构建、执行、部署-编译器-自动构建项目，这时候DEBUG running是不会自动构建的
    2. 高级设置 勾选 compiler running中构建， 这时候就成功了
