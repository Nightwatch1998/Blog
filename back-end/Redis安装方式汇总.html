# Redis安装方式汇总

# Windows环境

# 安装包下载

windows版本redis下载（GitHub）：

https://github.com/tporadowski/redis/releases （推荐）

https://github.com/MicrosoftArchive/redis/releases

官网下载（无Windows版本）： https://redis.io/download

所有版本这里都有：https://download.redis.io/releases/（下载后是个Linux的压缩文件，需要下载、解压和编译）

# Windows系统zip方式

# 安装并启动

解压压缩包，cmd进入解压目录，运行命令

redis-server.exe redis.windows.conf

出现redis图案即为启动成功

为了方便使用，可以把redis路径配置到path中

# 测试

使用安装目录的redis-cli.exe工具

redis-cli.exe -h 127.0.0.1 -p 6379
127.0.0.1:6379> ping
PONG

切换数据库

127.0.0.1:6379> select 8
OK
127.0.0.1:6379[8]> set name dish
OK
127.0.0.1:6379[8]> get name
"dish"
127.0.0.1:6379[8]> shutdown
not connected>

# Windows系统msi方式

# 安装

下载msi安装包后双击，设置安装位置和端口

在安装目录配置redis.windows-service.conf文件，找到requirepass foobared，在后面添加一行

requirepass 123456

配置连接密码

启动服务

在windows服务中查找redis，右键启动

E:\Program Files x86\redis>redis-cli
127.0.0.1:6379>

密码验证

127.0.0.1:6379> auth 123456
OK

验证get set

127.0.0.1:6379> set name dish
OK
127.0.0.1:6379> get name
"dish"

# 常用指令

安装为系统服务：
redis-server --service-install
卸载服务：
redis-server --service-uninstall
开启服务：
redis-server --service-start
停止服务：
redis-server --service-stop

# Linux系统

# 安装包下载

这里准备了redis-5.0.8.tar.gz，直接放到root目录

# debain系统

# 安装

1. 在/usr/local/下创建redis文件夹并进入
cd /usr/local/
mkdir redis
cd redis

2. 将redis安装包解压到/usr/local/redis
tar -zxvf /root/redis-5.0.8.tar.gz -C ./
解压后/usr/local/redis 出现redis-5.0.8目录

3. 进入目录编译
cd redis-5.0.8/
make install

4. 将redis安装未系统服务并后台启动
cd utils/
./install_server.sh
选择全部默认配置

5. 查看redis启动结果
systemctl status redis_6379

6. 启动redis-li测试
redis-cli

# 更改配置

远程访问

vim /etc/redis/6379.conf
bind 127.0.0.2
改为
bind 0.0.0.0
重启服务
systemctl restart redis_6379

设置访问密码

vim /etc/redis/6379.conf
找到如下内容
# requirepass foorbared
去掉注释，更换为自己的密码
requirepass huyadish
保存，重启服务
systemctl restart redis_6379
