# NodeJS安装配置

# windows环境

官网下载安装包，一路next

# linux环境

# 准备安装包

命令行直接下载

wget https://nodejs.org/dist/v16.13.1/node-v16.13.1-linux-x64.tar.xz

可以在官网仓库查看安装包对应的平台和版本

# 安装

创建目录

cd /usr/local
mkdir node
cd node

解压到当前目录

tar -xvfJ /root/node-v16.13.1-linux-x64.tar.xz

解压完毕，当前目录会出现node-v16.13.1-linux-x64目录

# 配置系统环境变量

打开配置文件

vi /etc/profile

末尾添加

export PATH=/usr/local/lib/node/node-v16.13.1-linux-x64/bin:$PATH

保存退出

刷新环境变量

source /etc/profile

# 验证

node -v
npm version
npx -v
